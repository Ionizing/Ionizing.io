多原子分子体系S-方程的描述

Hamil onian 算符

在非相对论近似下，分子体系满足多体 Schrodinger 方程： $$ \ha { H } \Phi = \varepsilon \Phi $$ 其中 $\ha {H}$ 是分子体系的 Hamil onian 算符，$\Phi$ 是分子体系的波函数。

设一个分子体系含 $M$ 个核，$N$ 个电子，则它的 Hamil onian 可以写成

$$ \begin{align} \ha { H } ={}& \ha { T } _ { e } + \ha { T } _ { N } + \ha { V } _ { N e } + \ha { V } _ { e e } + \ha { V } _ { N N }
={}& - \sum _ { i = 1 } ^ { N } \frac { 1 } { 2 } \nabla _ { i } ^ { 2 } - \sum _ { \alpha = 1 } ^ { M } \frac { 1 } { 2 M _ { \alpha } } \nabla _ { \alpha } ^ { 2 } - \sum _ { i = 1 } ^ { N } \sum _ { \alpha = 1 } ^ { M } \frac { Z _ { \alpha } } { r _ { i \alpha } } + \sum _ { i = 1 } ^ { N } \sum _ { j > i } ^ { N } \frac { 1 } { r _ { i j } } + \sum _ { \alpha = 1 } ^ { M } \sum _ { \be a > \alpha } ^ { M } \frac { Z _ { \alpha } Z _ { \be a } } { R _ { \alpha \be a } } \end{align} \ ag{2} $$

式子中 $\alpha, \be a$ 表示原子核的标号，$Z{\alpha}$ 表示核电荷数，$M{\alpha}$ 表示原子核的质量（原子单位制）， $i,j$ 表示电子的标号。

Born-Oppenheimer 近似

Born-Oppenheimer 近似：电子的质量远远小于核的质量，故核的运动相对于电子是相当缓慢的，因此研究电子的运动行为时认为核是不动的，就好比放牛时牛和它周围的牛蝇运动一样。

经过 Born-Oppenheimer 近似，电子与核的运动被解耦，它们被分开处理，分子的波函数可以写成电子波函数和核波函数乘积的形式，即 $$ \Phi = \Psi _ { e l } \Psi _ { N } \no ag $$ 于是电子运动和核的运动分别描述为 $$ \begin{cases} \lef ( \ha { T } _ { e } + \ha { V } _ { N e } + \ha { V } _ { e e } \righ ) \Psi _ { e l } = E _ { e l } \Psi _ { e l }
\lef (\ha { T } _ { N } + \ha { V } _ { N N }\righ ) \Psi _ { N } = \lef ( \varepsilon - E _ { e l } \righ ) \Psi _ { N } \end{cases} \label{eq:oppenheimer} $$

$\eqref{eq:oppenheimer}$ 式中第一个方程即为电子的运动方程，也是量子化学主要研究的问题。

平均场近似

忽略电子之间的瞬时运动关联，每个电子视为在核与其他电子建立的平均势场中运动。每个电子的状态用一个分子轨道（分子波函数）描述，即 $$ \ha { h } ( 1 ) \psi _ { i } ( 1 ) = \varepsilon _ { i } \psi _ { i } ( 1 ) \label{eq:mo} $$ 其中 ${ \psi_i }$ 为分子轨道（MO）； ${ \epsilon_i }$ 为轨道能 。

其中单电子算符 $ \ha { h } ( 1 ) = - \frac { 1 } { 2 } \nabla _ { 1 } ^ { 2 } + \sum _ { a } - \frac { Z _ { a } } { r _ { 1 a } } + \ha { v } _ { e f f } ( 1 ) $ ， $\ha { v } _ { e f f } ( 1 )$ 是等效单电子势，即其他电子对该电子相互作用的等效作用势。

总电子的波函数用行列式波函数（Sla er 行列式）描述，总电子能量为

$$ E _ { e l } = \lef \langle \Psi \lef | H _ { e l } \righ | \Psi \righ \rangle $$

LCAO 近似

分子轨道取为某些合理的原子轨道线性组合： $$ \psi _ { i } = \sum _ { v } c _ { v i } \varphi _ { v } \label{eq:lcao} $$ $\eqref{eq:lcao}$ 中$c_{vi}$ 表示原子轨道的系数， $\varphi_v$ 表示原子轨道。

注：这里的分子轨道指自旋轨道，区别于空间轨道。（自旋轨道 = 空间轨道 * 自旋， $\psi_i = \varphi_i \e a_i$）

行列式波函数

由于电子具有交换反对称性，简单乘积波函数 $$ \Phi = \prod_i \psi_i $$ 不能描述这个性质，故必须使用 Sla er 行列式描述体系的波函数，即： $$ \begin{align} \Psi \lef ( q _ { 1 } , \cdo s , q _ { N } \righ ) ={}& \frac { 1 } { \sqr { N ! } } \nonumber \begin{vma rix} \begin{array} { c c c } { \psi _ { 1 } \lef ( q _ { 1 } \righ ) } & { \cdo s } & { \psi _ { 1 } \lef ( q _ { N } \righ ) } \ { \psi _ { 2 } \lef ( q _ { 1 } \righ ) }& \cdo s & { \psi _ { 2 } \lef ( q _ { N } \righ ) } \ { \vdo s }& \ddo s & { \vdo s } \ { \psi _ { N } \lef ( q _ { 1 } \righ ) }& \cdo s & { \psi _ { N } \lef ( q _ { N } \righ ) } \end{array} \end{vma rix} \ \nonumber ={}& | \psi _ { 1 } \psi _ { 2 } \cdo s \psi _ { N } \rangle
={}& \frac { 1 } { \sqr { N ! } } \sum _ { n = 1 } ^ { N ! } ( - 1 ) ^ { P _ { n } } \ha { P } _ { n } \lef { \psi _ { 1 } \lef ( q _ { 1 } \righ ) \psi _ { 2 } \lef ( q _ { 2 } \righ ) \cdo s \psi _ { N } \lef ( q _ { N } \righ ) \righ } \label{eq:sla er} \end{align} $$

  • $\ha {P}$ 算符是交换算符， $\ha {P}_{12}$ 表示将 1, 2 号轨道上的电子互换；
  • $\eqref{eq:sla er}$ 中的每个电子的分子轨道都满足正交归一性： $ \lef \langle \psi _ { i } | \psi _ { j } \righ \rangle = \in \psi _ { i } ^ { * } ( 1 ) \psi _ { j } ( 1 ) d q _ { 1 } = \del a _ { i j } $ （即每个电子只能占据一个轨道）；
  • 如果两个行列式波函数的自旋轨道不完全相同，则它们正交。

库仑作用与交换作用

库仑作用

库仑作用是指两个电子（无论自旋是否相同）之间存在的静电相互作用： $$ \begin{aligned} J _ { 1 s 2 s } & = \in \frac { \lef | \psi _ { 1 } ( 1 ) \righ | ^ { 2 } \cdo \lef | \psi _ { 2 } ( 2 ) \righ | ^ { 2 } } { r _ { 12 } } d q _ { 1 } d q _ { 2 } \ & = \in \frac { \lef | \varphi _ { 1 s } ( 1 ) \righ | ^ { 2 } \cdo \varphi _ { 2 s } \lef . ( 2 ) \righ | ^ { 2 } } { r _ { 12 } } d \vec { r } _ { 1 } d \vec { r } _ { 2 } \in \lef | \e a _ { 1 } ( 1 ) \righ | ^ { 2 } d \omega _ { 1 } \in \lef | \e a _ { 2 } ( 2 ) \righ | ^ { 2 } d \omega _ { 2 } \end{aligned} $$

交换作用

以 He​ 体系为例： $$ \Psi ( 1,2 ) = \frac { 1 } { \sqr { 2 } } \lef | \begin{array} { l l } { \psi _ { 1 } ( 1 ) } & { \psi _ { 1 } ( 2 ) } \ { \psi _ { 2 } ( 1 ) } & { \psi _ { 2 } ( 2 ) } \end{array} \righ | $$ 则它的能量期待值为： $$ \begin{aligned} E ={}& \frac { 1 } { 2 } \lef \langle \psi _ { 1 } ( 1 ) \psi _ { 2 } ( 2 ) - \psi _ { 2 } ( 1 ) \psi _ { 1 } ( 2 ) \lef | \ha { h } ( 1 ) + \ha { h } ( 2 ) + \frac { 1 } { r _ { 12 } } \righ | \psi _ { 1 } ( 1 ) \psi _ { 2 } ( 2 ) - \psi _ { 2 } ( 1 ) \psi _ { 1 } ( 2 ) \righ \rangle
={}& \frac { 1 } { 2 } \lef { 2 \ imes \lef ( \varepsilon _ { 1 s } ^ { 0 } + \varepsilon _ { 2 s } ^ { 0 } + J _ { 1 s 2 s } \righ ) - 2 \ imes \lef [ \lef \langle \psi _ { 1 } ( 1 ) | \ha { h } ( 1 ) | \psi _ { 2 } ( 1 ) \righ \rangle \lef \langle \psi _ { 2 } ( 2 ) | \psi _ { 1 } ( 2 ) \righ \rangle +\righ . \righ .
& \lef . \lef . \lef \langle \psi _ { 1 } ( 1 ) | \psi _ { 2 } ( 1 ) \righ \rangle \lef \langle \psi _ { 2 } ( 2 ) | \ha { h } ( 2 ) | \psi _ { 1 } ( 2 ) \righ \rangle + \lef \langle \psi _ { 1 } ( 1 ) \psi _ { 2 } ( 2 ) \lef | \frac { 1 } { r _ { 12 } } \righ | \psi _ { 2 } ( 1 ) \psi _ { 1 } ( 2 ) \righ \rangle \righ ] \righ }
={}& \frac { 1 } { 2 } \lef { 2 \ imes \lef ( \varepsilon _ { 1 s } ^ { 0 } + \varepsilon _ { 2 s } ^ { 0 } + J _ { 1 s 2 s } \righ ) - 2 \ imes \lef [ 0 + 0 + \lef \langle \psi _ { 1 } ( 1 ) \psi _ { 2 } ( 2 ) \lef | \frac { 1 } { r _ { 12 } } \righ | \psi _ { 2 } ( 1 ) \psi _ { 1 } ( 2 ) \righ \rangle \righ ] \righ }
={}& \varepsilon _ { 1 s } ^ { 0 } + \varepsilon _ { 2 s } ^ { 0 } + J _ { 1 s 2 s } - K _ { 1 s 2 s } \end{aligned} $$ 上式中的 $K_{1s 2s} = \lef \langle \psi _ { 1 } ( 1 ) \psi _ { 2 } ( 2 ) \lef | \frac { 1 } { r _ { 12 } } \righ | \psi _ { 2 } ( 1 ) \psi _ { 1 } ( 2 ) \righ \rangle$ 即为交换能。

交换作用是指同自旋的两个电子之间存在的相互作用： $$ \begin{aligned} K _ { 1s 2s } ={}& \in \frac { \psi_ { 1 } ( 1 ) ^* \psi _ { 2 } ( 2 ) ^* \psi _ { 2 } ( 1 ) \psi _ { 1 } ( 2 ) } { r _ { 12 } } d q _ { 1 } d q _ { 2 }
={} & \in \frac { \varphi _ { 1 s } ( 1 ) ^ { * } \varphi _ { 2 s } ( 2 ) ^* \varphi _ { 2 s } ( 1 ) \varphi _ { 1 s } ( 2 ) } { r _ { 12 } } d \vec { r _ { 1 } } d \vec { r _ { 2 } } \in \e a _ { 1 } ( 1 ) ^ { * } \e a _ { 2 } ( 1 ) d \omega _ { 1 } \in \e a _ { 2 } ( 2 ) ^ { * } \e a _ { 1 } ( 2 ) d \omega _ { 2 } \end{aligned} $$

  • 当 $\psi_1, \psi_2$ 同自旋， $\e a_1 = \e a_2$ ，$K \neq 0$ ($K>0$)；

  • 当 $\psi_1, \psi_2$ 反自旋， $\e a_1 \neq \e a_2$ ，$K = 0$ 。

这一项可以通过求 Sla er 行列式的能量期待值得到。

注：$J{11} - K{11} = 0$ ，电子不存在自相关作用。

行列式波函数的 Pros

  1. 行列式波函数部分地考虑了同自旋电子间的运动关联（不允许电子占据同一个自旋轨道）；
  2. 同自旋电子间存在交换作用，它使总能量降低；
  3. 多电子原子中，$n, l$ 相同的简并轨道上的电子，将分占磁量子数 $m$ 不同的轨道，并使其自旋平行（Hund 规则）

Sla er-Condon 规则

Sla er-Condon 规则即为 Sla er 行列式矩阵元的计算规则，

$$ \lef \langle \Psi ^ { \prime } | \ha { O } | \Psi \righ \rangle = \in \lef [ \lef ( \Psi ^ { \prime } \righ ) ^* \ha { O } \Psi \righ ] d q _ { 1 } \cdo s d q _ { N } = ? $$

准备

  1. 算符：

    1. 单电子算符：$ \ha { O } _ { 1 } = \sum _ { n } \ha { h } ( n ) $ ；
    2. 双电子算符： $ \ha { O } _ { 2 } = \sum _ { n < m } \frac { 1 } { r _ { n m } } $ 。

  2. 行列式：

    1. 对角矩阵元： $$ | \Psi ^ { \prime } \rangle = | \Psi \rangle \nonumber $$

    2. 非对角矩阵元 1: $$ \begin{aligned} | \Psi \rangle & = | \psi _ { 1 } \cdo s \psi _ { k } \cdo s \psi _ { N } \rangle \ | \Psi ^ { \prime } \rangle & = | \psi _ { 1 } \cdo s \psi _ { a } \cdo s \psi _ { N } \rangle \quad \ ex {单激发组态} \end{aligned} $$

    3. 非对角矩阵元 2: $$ \begin{array} { l } { | \Psi \rangle = | \cdo s \psi _ { k } \cdo s \psi _ { l } \cdo s \rangle } \ { | \Psi ^ { \prime \prime } \rangle = | \do s \psi _ { a } \cdo s \psi _ { b } \cdo s \rangle } \quad \ ex {双激发组态} \end{array} \nonumber $$

规则

  1. 重叠：

    1. 对角矩阵元： $$ \langle \Psi | \Psi \rangle = 1 $$

    2. 单激发态&双激发态： $$ \lef \langle \Psi ^ { \prime } | \Psi \righ \rangle = 0 $$

  2. 单电子算符：

    1. 对角矩阵元： $$ \lef \langle \Psi \lef | \ha { O } _ { 1 } \righ | \Psi \righ \rangle = \sum _ { i } \lef \langle \psi _ { i } | \ha { h } | \psi _ { i } \righ \rangle = \sum _ { i } \langle i | \ha { h } | i \rangle $$

    2. 单激发态： $$ \lef \langle \Psi ^ { \prime } \lef | \ha { O } _ { 1 } \righ | \Psi \righ \rangle = \lef \langle \psi _ { a } | \ha { h } | \psi _ { k } \righ \rangle = \langle a | \ha { h } | k \rangle $$

    3. 双激发态： $$ \lef \langle \Psi ^ { \prime } \lef | \ha { O } _ { 1 } \righ | \Psi \righ \rangle = 0 $$

  3. 双电子算符：

    1. 对角矩阵元： $$ \begin{aligned} \lef \langle \Psi \lef | \ha { O } _ { 2 } \righ | \Psi \righ \rangle & = \frac { 1 } { 2 } \sum _ { i = 1 } ^ { N } \sum _ { j = 1 } ^ { N } \lef \langle \psi _ { i } \psi _ { j } | \psi _ { i } \psi _ { j } \righ \rangle - \lef \langle \psi _ { i } \psi _ { j } | \psi _ { j } \psi _ { i } \righ \rangle \ & = \frac { 1 } { 2 } \sum _ { i = 1 } ^ { N } \sum _ { j = 1 } ^ { N } \langle i j | i j \rangle - \langle i j | j i \rangle \end{aligned} $$

    2. 单激发态： $$ \lef \langle \Psi ^ { \prime } \lef | \ha { O } _ { 2 } \righ | \Psi \righ \rangle = \sum _ { i = 1 } ^ { N } \langle i a | i k \rangle - \langle i a | k i \rangle $$

    3. 双激发态： $$ \lef \langle \Psi ^ { \prime } \lef | \ha { O } _ { 2 } \righ | \Psi \righ \rangle = \langle a b | k l \rangle - \langle a b | l k \rangle $$

Ha ree-Fock 方程

Ha ree-Fock 方程（非正则形式）： $$ \lef [ \ha { h } ( 1 ) + \sum _ { j = 1 } ^ { N } \lef ( \ha { J } _ { j } ( 1 ) - \ha { K } _ { j } ( 1 ) \righ ) \righ ] \psi _ { i } ( 1 ) = \sum _ { j = 1 } ^ { N } \varepsilon _ { i j } \psi _ { j } ( 1 ) \label{eq:hf_ununi ary} $$ 其中，库伦算符和交换算符： $$ \begin{align} \ha { J } _ { j } ( 1 ) \psi _ { i } ( 1 ) = \lef [ \in \psi _ { j } ^ { * } ( 2 ) \frac { 1 } { r _ { 12 } } \psi _ { j } ( 2 ) d q _ { 2 } \righ ] \psi _ { i } ( 1 )
\ha { K } _ { j } ( 1 ) \psi _ { i } ( 1 ) = \lef [ \in \psi _ { j } ^ { * } ( 2 ) \frac { 1 } { r _ { 12 } } \psi _ { i } ( 2 ) d q _ { 2 } \righ ] \psi _ { j } ( 1 ) \end{align} $$ $\eqref{eq:hf_ununi ary}$ 中 $\ha { h } ( 1 ) + \sum _ { j = 1 } ^ { N } \lef ( \ha { J } _ { j } ( 1 ) - \ha { K } _ { j } ( 1 ) \righ )$ 可用 Fock 算符 $\ha {f}(1)$ 简写代替。

经过 Uni ary 变换， $\eqref{eq:hf_ununi ary}$ 可以变换为正则形式，即： $$ \ha { f } ( 1 ) \psi _ { i } ( 1 ) = \varepsilon _ { i } \psi _ { i } ( 1 ) $$

注：$\eqref{eq:fock}$ 中的 $\ha { h } ( 1 ) = - \frac { 1 } { 2 } \nabla _ { 1 } ^ { 2 } - \sum _ { \alpha } \frac { Z _ { \alpha } } { r _ { 1 \alpha } } $ 表示核 Hamil onian 项，是单电子算符，与 $\eqref{eq:mo}$ 中 $\ha {h}(1)$ 的含义有所区别，请注意区分。

Fock 算符的性质

Fock 算符： $$ \ha {f}(1) = \ha { h } ( 1 ) + \sum _ { j = 1 } ^ { N } \lef ( \ha { J } _ { j } ( 1 ) - \ha { K } _ { j } ( 1 ) \righ ) \label{eq:fock} $$

  • Fock 算符是等效单电子的哈密顿算符，它的本征函数式分子轨道，本征值为轨道能；
  • Fock 算符是 Hermi ian 算符（因为交换算符和库仑算符都是 Hermi ian 算符，动能项以及核势能项也都是 Hermi ian 算符）；
  • Fock 算符是分子点群的对称算符，分子轨道属于分子点群的不可约表示；
  • Fock 算符之和 $\sum _ { n = 1 } ^ { N } \ha { f } ( n ) = \ha { H } ^ { H F } \neq \ha { H } _ { e l }$ ，它将电子间的作用重复计入。

轨道能与电子总能量

轨道能

$$ \varepsilon _ { i } = \lef \langle \psi _ { i } | \ha { f } | \psi _ { i } \righ \rangle = \varepsilon _ { i } ^ { 0 } + \sum _ { j } ^ { N } \lef ( J _ { i j } - K _ { i j } \righ ) \label{eq:orb_en} $$

$\eqref{eq:orb_en}$ 中：

  • $\varepsilon_i^0$ 为 $\ha {h}(1)$ 的本征值， $\varepsilon_i$ 为第 $i$ 个轨道的轨道能；
  • $J{ij}=\langle ij|ij\rangle = \in dq{1}dq_{2}\psi _{i}^{\s ar}(1)\psi {j}^{\s ar}(2)\frac {1}{r{12}}\psi _{i}(1)\psi _{j}(2)$ ；
  • $K{ij} = \langle ij|ji \rangle = \in dq{1}dq_{2} \psi _ { i } ^ { \s ar } ( 1 ) \psi _ { j } ^ { \s ar } ( 2 ) \frac { 1 } { r _ { 12 } } \psi _ { j } ( 1 ) \psi _ { i } ( 2 )$ 。

电子总能量

$$ E _ { 0 } = \sum _ { i } ^ { N } \varepsilon _ { i } ^ { 0 } + \frac { 1 } { 2 } \sum _ { i } ^ { N } \sum _ { j } ^ { N } \lef [ J _ { i j } - K _ { i j } \righ ] \label{eq:ele_en} $$

$\eqref{eq:ele_en}$ 中第二项的 $\frac{1}{2}$ 表示去除因双重求和重复计入的交换能和库仑能 。

注：分子轨道能之和不等于电子总能 $$ \sum _ { i } ^ { N } \varepsilon _ { i } = \sum _ { i } ^ { N } \varepsilon _ { i } ^ { 0 } + \sum _ { i } ^ { N } \sum _ { j } ^ { N } \lef [ J _ { i j } - K _ { i j } \righ ] \neq E _ { 0 } $$

Koopmans 定理

假如从占据轨道 $\psi_k$ 上电离一个电子而其他轨道不变，则分子的电离势为 $-\varepsilon_k$ 。假定在空轨道 $\psi_r$ 上亲和一个电子而其他轨道不变，则分子的电子亲和势为 $\varepsilon_r$ ，简言之：当不考虑轨道弛豫时电离能/亲和能近似等于被电离/亲和电子的轨道能。

电离能

电离前：

$$ \begin{align} &| \Psi _ { 0 } ^ { N } \rangle = | \psi _ { 1 } \cdo s \psi _ { k } \cdo s \psi _ { N } \rangle \nonumber
&E _ { 0 } = \lef \langle \Psi _ { 0 } ^ { N } | \ha { H } | \Psi _ { 0 } ^ { N } \righ \rangle = \sum _ { i } ^ { N } \varepsilon _ { i } ^ { 0 } + \frac { 1 } { 2 } \sum _ { i } \sum _ { j } \lef ( J _ { i j } - K _ { i j } \righ ) \nonumber \end{align} $$

电离后： $$ \begin{align} | \Psi _ { k } ^ { N - 1 } \rangle ={}& | \psi _ { 1 } \cdo s \psi _ { k - 1 } \psi _ { k + 1 } \cdo s \psi _ { N } \rangle \nonumber
E _ { k } ^ { + } ={}& \lef \langle \Psi _ { k } ^ { N - 1 } | \ha { H } | \Psi _ { k } ^ { N - 1 } \righ \rangle = \sum _ { i \neq k } \varepsilon _ { i } ^ { 0 } + \frac { 1 } { 2 } \sum _ { i \neq k } \sum _ { j \neq k } \lef ( J _ { i j } - K _ { i j } \righ ) \nonumber \end{align} $$

则电离能： $$ \begin{align} I.P. ={}& E _ { k } ^ { + } - E _ { 0 }
={}& - \varepsilon _ { k } ^ { 0 } - \frac { 1 } { 2 } \lef [ \sum _ { j } \lef ( J _ { k j } - K _ { k j } \righ ) + \sum _ { i } \lef ( J _ { i k } - K _ { i k } \righ ) \righ ]
={}& - \varepsilon _ { k } ^ { 0 } - \sum _ { j } \lef ( J _ { k j } - K _ { k j } \righ ) = - \varepsilon _ { k } \end{align} $$

亲和能

与电离能同理， $$ E . A . = E _ { 0 } - E _ { r } ^ { N + 1 } = - \varepsilon _ { r } ^ { 0 } - \sum _ { j } \lef ( J _ { r j } - K _ { j r } \righ ) = - \varepsilon _ { r } $$

Koopmans 定理提供了一种近似求解 $I.P$ 与 $E.A.$ 的方法，但它不考虑电离/亲和后的轨道弛豫和电子的相关效应（事实上整个 Ha ree-Fock 近似也没有考虑相关效应），这样求得的电离能与亲和能有误差。

Roo han 方程

Roo han 方程的本质是将 Ha ree-Fock 方程的分子轨道用一系列基函数展开得到的方程。

限制性与非限制性自旋轨道

限制性自旋轨道：假设同一个分子轨道分为两个自旋相反的自旋轨道，它们的空间轨道部分完全相同。限制性自旋轨道一般用于描述闭壳层的电子行为。 $$ \psi _ { a } \lef ( q _ { 1 } \righ ) \longrigh arrow \lef { \begin{array} { l } { \varphi _ { i } \lef ( \vec { r } _ { 1 } \righ ) \alpha \lef ( \omega _ { 1 } \righ ) } \ { \varphi _ { i } \lef ( \vec { r } _ { 1 } \righ ) \be a \lef ( \omega _ { 1 } \righ ) } \end{array} \righ . $$ 非限制性自旋轨道：同一个分子轨道中自旋轨道的空间轨道部分不同。 $$ \psi _ { a } \lef ( q _ { 1 } \righ ) \longrigh arrow \lef { \begin{array} { l } { \varphi _ { i } \lef ( \vec { r } _ { 1 } \righ ) \alpha \lef ( \omega _ { 1 } \righ ) } \ { \varphi _ { i } ^ {\color{red} \prime } \lef ( \vec { r } _ { 1 } \righ ) \be a \lef ( \omega _ { 1 } \righ ) } \end{array} \righ . $$ 非限制性自旋轨道一般用于描述开壳层的电子行为。

空间轨道的 HF 方程

空间轨道的 Ha ree-Fock 方程为： $$ \ha { f } \lef ( \vec { r } _ { 1 } \righ ) \varphi _ { i } \lef ( \vec { r } _ { 1 } \righ ) = \varepsilon _ { i } \varphi _ { i } \lef ( \vec { r } _ { 1 } \righ ) \label{eq:hf_space} $$ 其中空间轨道的 Fock 算符为： $$ \begin{aligned} { f \lef ( \vec { r } _ { 1 } \righ ) = \ha { h } \lef ( \vec { r } _ { 1 } \righ ) + \sum _ { j = 1 } ^ { N / 2 } 2 \ha { J } _ { j } \lef ( \vec { r } _ { 1 } \righ ) - \ha { K } _ { j } \lef ( \vec { r } _ { 1 } \righ ) } \ { \ha { J } _ { j } \lef ( \vec { r } _ { 1 } \righ ) \varphi _ { i } \lef ( \vec { r } _ { 1 } \righ ) = \lef [ \in \frac { \varphi _ { j } ^ { * } \lef ( \vec { r } _ { 2 } \righ ) \varphi _ { j } \lef ( \vec { r } _ { 2 } \righ ) } { r _ { 12 } } d \vec { r } _ { 2 } \righ ] \varphi _ { i } \lef ( \vec { r } _ { 1 } \righ ) } \ { \ha { K } _ { j } \lef ( \vec { r } _ { 1 } \righ ) \varphi _ { i } \lef ( \vec { r } _ { 1 } \righ ) = \lef [ \in \frac { \varphi _ { j } \lef ( \vec { r } _ { 2 } \righ ) \varphi _ { i } \lef ( \vec { r } _ { 2 } \righ ) } { r _ { 12 } } d \vec { r _ { 2 } } \righ ] \varphi _ { j } \lef ( \vec { r } _ { 1 } \righ ) } \end{aligned} $$ 上面式子中 $ \ha {K} $ 项前面没有 2 是因为该波函数为限制性自旋波函数，体系为闭壳层，自旋向上和自旋向下的电子个数相同，恰好有一半的空间轨道有电子占据，且这些空间轨道都已经占满，故只有自旋平行的电子之间有交换作用（即只有 $\frac{N}{2}$ 个电子之间有交换作用）。

分子轨道的轨道能（自旋轨道）： $$ \varepsilon_i = \varepsilon _ { i } ^ { 0 } + \sum _ { j } ^ { N } \lef ( J _ { i j } - K _ { i j } \righ ) $$ 分子轨道的轨道能（空间轨道，闭壳层限制性）： $$ \varepsilon_i = \varepsilon _ { i } ^ { 0 } + \sum _ { j } ^ { N / 2 } \lef ( 2 J _ { i j } - K _ { i j } \righ ) $$ 电子总能量（自旋轨道）： $$ E_0= \sum _ { i } ^ { N } h _ { i i } + \frac { 1 } { 2 } \sum _ { i } ^ { N } \sum _ { j } ^ { N } J _ { i j } - K _ { i j } $$ 电子总能量（空间轨道，闭壳层限制性）： $$ E _ 0 = \sum _ { i } ^ { N / 2 } 2 h _ { i i } + \frac { 1 } { 2 } \sum _ { i } ^ { N / 2 } \sum _ { j } ^ { N / 2 } 2 J _ { i j } - K _ { i j } $$ 上式中第二项前的 $\frac{1}{2}$ 表示已经减去重复计入的空间轨道的相互作用。

Roo haan 方程

引入一组已知的基函数将空间轨道展开： $$ \varphi _ { i } = \sum _ { v = 1 } ^ { m } C _ { vi } \chi _ { v } \quad i = 1,2 , \cdo s , m \label{eq:roo haan_base} $$ 将 $\eqref{eq:roo haan_base}$ 代入 $\eqref{eq:hf_space}$ 可以得到： $$ \ha { f } \lef ( \vec { r } _ { 1 } \righ ) \varphi _ { i } \lef ( \vec { r } _ { 1 } \righ ) = \varepsilon _ { i } \varphi _ { i } \lef ( \vec { r } _ { 1 } \righ ) \quad i = 1,2 , \cdo s , m $$ 上式左乘 $\chi_{\mu}^* (1)$ 并积分，得到： $$ \sum _ { v } ^ { m } C _ { v i } \langle \chi _ { \mu } ( 1 ) | \ha { f } ( 1 ) | \chi _ { v } ( 1 ) \rangle = \varepsilon _ { i } \sum _ { v } ^ { m } C _ { v i } \langle \chi _ { \mu } ( 1 ) | \chi _ { v } ( 1 ) \rangle \quad i = 1,2 , \cdo s , m $$ 令 $ S _ { \mu \nu } = \langle \chi _ { \mu } ( 1 ) | \chi _ { v } ( 1 ) \rangle = \in d \vec { r } _ { 1 } \chi _ { \mu } ^ { * } ( 1 ) \chi _ { v } ( 1 ) $ ，

$ F _ { \mu v } = \langle \chi _ { \mu } ( 1 ) | \ha { f } ( 1 ) | \chi _ { v } ( 1 ) \rangle = \in d \vec { r } _ { 1 } \chi _ { \mu } ^ { * } ( 1 ) \ha { f } ( 1 ) \chi _ { v } ( 1 ) $

由此可以得到方程组： $$ \sum _ { v } ^ { m } F _ { \mu \nu } C _ { v i } = \varepsilon _ { i } \sum _ { v } ^ { m } S _ { \mu \nu } C _ { v i } \quad i = 1,2 , \cdo s , m $$ 或者写成矩阵形式： $$ \ma hbf { F C } _ { i } = \varepsilon _ { i } \ma hbf { S } \ma hbf { C } _ { i } \quad i = 1,2 , \cdo s , m $$ 合并写成矩阵方程： $$ \ma hbf { F C } = \ma hbf { S } \ma hbf { C } \boldsymbol { \varepsilon } \ ag{*} \label{eq:roo haan} $$ $\eqref{eq:roo haan}​$ 即为 Roo haan 方程。

其中： $$ \ma hbf { C } = \lef ( \begin{array} { c c c c } { C _ { 11 } } & { C _ { 12 } } & { \cdo s } & { C _ { 1 m } } \ { C _ { 21 } } & { C _ { 22 } } & { \cdo s } & { C _ { 2 m } } \ { C _ { 31 } } & { C _ { 32 } } & { \cdo s } & { C _ { 3 m } } \ { \vdo s } & { \vdo s } & { } & { \vdo s } \ { C _ { m 1 } } & { C _ { m 2 } } & { \cdo s } & { C _ { m m } } \end{array} \righ ) ,\quad \boldsymbol { \varepsilon } = \lef ( \begin{array} { c c c c } { \varepsilon _ { 1 } } & { } & { } & { O } \ { } & { \varepsilon _ { 2 } } & { } & { } \ { } & { } & { \ddo s } & { } \ { O } & { } & { } & { \varepsilon _ { m } } \end{array} \righ ) $$ 它们分别表示分子轨道（空间轨道）和轨道能。

$C_{ij}$ 表示第 $j$ 个分子轨道（空间轨道）第 $i$ 个基函数的系数。

重叠矩阵

$$ S _ { \mu \nu } = \langle \chi _ { \mu } ^ { * } ( 1 ) | \chi _ { v } ( 1 ) \rangle = \in d \vec { r } _ { 1 } \chi _ { \mu } ^ { * } ( 1 ) \chi _ { v } ( 1 ) $$

且 $$ S _ { \mu \nu } ^ { * } = \lef [ \in d \vec { r } _ { 1 } \chi _ { \mu } ^ { * } ( 1 ) \chi _ { v } ( 1 ) \righ ] ^ { * } = \in d \vec { r } _ { 1 } \chi _ { v } ^ { * } ( 1 ) \chi _ { \mu } ( 1 ) = S _ { v \mu } $$ 故 $\ma hbf{S}$ 为一个 $m \ imes m$ 的 Hermi ian 矩阵，且 $0 \leq \lef | S _ { \mu\nu } \righ | \leq 1$

Fock矩阵

$$ \begin{aligned} \ma hbf{F} _ { \mu \nu } ={}& \langle \chi _ { \mu } ^ { * } ( 1 ) \ha { f } ( 1 ) \chi _ { v } ( 1 ) \rangle
={}& \in d \vec { r } _ { 1 } \chi _ { \mu } ^ { * } ( 1 ) \ha { f } ( 1 ) \chi _ { v } ( 1 )
={} & h _ { \mu v } ^ { c o r e } + \sum _ { j } ^ { N / 2 } \in d \vec { r } _ { 1 } \chi _ { \mu } ^ { * } ( 1 ) \lef [ 2 \ha { J } _ { j } ( 1 ) - \ha { K } _ { j } ( 1 ) \righ ] \chi _ { v } ( 1 ) \end{aligned} $$

令 $\varphi _ { j } = \sum _ { \sigma } ^ { m } C _ { \sigma j } \chi _ { \sigma }$ ，$ \varphi _ { j } ^ { * } = \sum _ { \lambda } ^ { m } C _ { \lambda j } ^ { * } \chi _ { \lambda } ^ { * } $ ，则 $$ \begin{aligned} \in \chi _ { \mu } ^ { * } ( 1 ) \ha { J } _ { j } ( 1 ) \chi _ { v } ( 1 ) d \vec { r } _ { 1 } = \sum _ { \lambda } ^ { m } \sum _ { \sigma } ^ { m } C _ { \sigma j } C _ { \lambda j } ^ { * } \langle \mu v | \lambda \sigma \rangle
\in \chi _ { \mu } ^ { * } ( 1 ) \ha { K } _ { j } ( 1 ) \chi _ { v } ( 1 ) d \vec { r } _ { 1 } = \sum _ { \lambda } ^ { m } \sum _ { \sigma } ^ { m } C _ { \sigma j } C _ { \lambda j } ^ { * } \langle \mu {\color{red} \sigma} | \lambda {\color{red} v} \rangle \end{aligned} $$ 所以 Fock 矩阵的矩阵元可以进一步写成 $$ \begin{aligned} F _ { \mu \nu } ={}& h _ { \mu \nu } ^ { c o r e } + \sum _ { \lambda , \sigma } ^ { m , m } \sum _ { j } ^ { N / 2 } C _ { \sigma j } C _ { \lambda j } ^ { * } [ 2 ( \mu v | \lambda \sigma ) - ( \mu \sigma | \lambda v ) ]

={}& h _ { \mu \nu } ^ { c o r e } + G _ { \mu \nu } \end{aligned} $$ $G{\mu\nu}$ 矩阵即为 Fock 矩阵的双电子项，由密度矩阵 $P{\sigma \lambda}$和基函数的双电子积分给出。

密度矩阵

定义密度矩阵： $$ P _ { \sigma \lambda } = \sum _ { j } ^ { N / 2 } C _ { \sigma j } C _ { \lambda j } ^ { * } = \sum _ { j } ^ { N / 2 } ( C ) _ { \sigma j } \lef ( C ^ { \dagger } \righ ) _ { j \lambda } $$ $C$ 是分子轨道基函数的展开系数。

由于闭壳层的限制性，只有 $N/2$ 个空间轨道占有电子，且都占满，故密度矩阵通常等于 $$ P = C _ { 0 } C _ { 0 } ^ { \dagger } = \lef ( \begin{array} { c c c } { \vdo s } & { } & { } \ { c _ { \sigma 1 } } & { \do s } & { c _ { \sigma N / 2 } } \ { \vdo s } & { } & { } \end{array} \righ ) \lef ( \begin{array} { c c c } { } & { c _ { \lambda 1 } ^ { * } } \ { \ldo s } & { \vdo s } & { \cdo s } \ { } & { c _ { \lambda N / 2 } ^ { * } } \end{array} \righ ) $$

基的幺正化

Roo haan 方程 $\ma hbf { F C } = \ma hbf { S } \ma hbf { C\varepsilon } $ 不是标准的矩阵本征值方程，如果 $\ma hbf{S} = 1$，则 $\ma hbf { F C } = \ma hbf { C } \varepsilon$ 是标准的本征值方程。所以要对 Roo haan 方程进行 Uni ary 变换使其满足标准本征值方程，才能进行求解。

构造变换矩阵 $\ma hbf{X}​$ 使得 $$ \ma hbf { X } ^ { \dagger } \ma hbf { S X } = \ma hbf { S } ^ { \prime } = \ma hbf { I } $$

则构造变换后的 Fock 矩阵 $\ma hbf{F}‘$

$$ \ma hbf { F } ^ { \prime } = \ma hbf { X } ^ { + } \ma hbf { F } \ma hbf { X } $$ 继续假设原系数矩阵 $\ma hbf { C } = \ma hbf { X } \ma hbf { C } ^ { \prime }$ ，此时得到变换后的 Roo haan 方程： $$ \begin{aligned} \ma hbf { F X C } ^ { \prime } ={}& \ma hbf { S X C ^ { \prime } \varepsilon }
\lef ( \ma hbf { X } ^ { + } \ma hbf { F } \ma hbf { X } \righ ) \ma hbf { C } ^ { \prime } ={}& \lef ( \ma hbf { X } ^ { + } \ma hbf { S X } \righ ) \ma hbf { C } ^ { \prime } \boldsymbol { \varepsilon }
\ma hbf { F } ^ { \prime } \ma hbf { C } ^ { \prime } ={}& \ma hbf { C } ^ { \prime } \boldsymbol { \varepsilon } \end{aligned} $$ 此时对角化 Fock 矩阵 $\ma hbf{F}‘$ ，即可得到系数矩阵 $\ma hbf C’$ 和本征值 $\boldsymbol{\varepsilon}$ 。

SCF 迭代步骤

  1. 给定分子的初始信息（核位置 $\vec{R}\alpha$，元素种类 ${Z{\alpha}}$ ，电子个数 $N_{elec}$ ，以及基函数 $\lef { \chi _ { v } \righ }$ ）；
  2. 计算分子积分 $S _ { \mu \nu } , \quad h _ { \mu \nu } ^ { \ ex { core } } , \langle \mu v | \lambda \sigma \rangle$ ；
  3. 幺正化 $\ma hbf{S}$ 矩阵，得到变换矩阵 $\ma hbf{X}$ ；
  4. 猜一个初始密度矩阵 $\ma hbf{P}$ ；
  5. 由密度矩阵 $\ma hbf{P}$ 和双电子积分计算 $\ma hbf{G}$ 矩阵；
  6. 构造 Fock 矩阵 $ \ma hbf { F } = \ma hbf { h } ^ { \ma hrm { core } } + \ma hbf { G } $ ；
  7. 计算变换后的 Fock 矩阵 $\ma hbf { F } ^ { \prime } = \ma hbf { X } ^ { + } \ma hbf { F } \ma hbf { X }$ ；
  8. 对角化 $\ma hbf{F}‘$ 得到 $\ma hbf{C}‘$ 和 $\boldsymbol{\varepsilon}$ ；
  9. 计算系数矩阵 $\ma hbf { C } = \ma hbf { X } \ma hbf { C } ^ { \prime }$ ；
  10. 构造密度矩阵 $\ma hbf{P}$ ： $P _ { \mu } = \sum _ { j } ^ { N / 2 } C _ { \mu } C _ { v j } ^ { * }$ ；
  11. 验证密度矩阵 $\ma hbf{P}$ 和能量 $\varepsilon$ 是否收敛，若未收敛，返回步骤 5，用刚得到的密度矩阵 $\ma hbf{P}$ 计算 $\ma hbf{G}$ 矩阵；
  12. 计算已经收敛，迭代结束，计算能量期待值与其它性质。

基本计算结果

  1. 轨道能与电子能量；
  2. 势能面、构型优化、振动频率分析；
  3. 单电子性质（如分子偶极矩）；
  4. 电荷密度计算等。

总结

Ha ree-Fock 方法的几个关键点：

  1. 使用 Sla er 行列式描述分子的波函数；
  2. 使用基函数将分子轨道展开，便于计算；
  3. 使用自洽场方法迭代计算；
  4. $J{11} - K{11} = 0$ ，没有自相关作用。
